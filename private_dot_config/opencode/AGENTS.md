# Agents

- If prompted to give information about a tool or a programming language, be sure to look through the relevant documentation for that language before answering.
- Be thorough, taking longer to answer by thinking more is better than giving a short incomplete wrong answer.
- Output plain Markdown only.
- No LaTeX, no `$...$`, no `\(...\)`, no `\[...\]`.
- Write math as plain text (`AoS -> SoA`, `x_i >= target`).
- Use normal fenced code blocks (```rust```, ```python```, ```bash```).
- Use simple lists and GitHub checkboxes (`- [ ] task`).
